# About
Sublime Text 2 Plugin - toggle boolean words

```
false <-> true
  yes <-> no
   on <-> off
    0 <-> 1
    + <-> -
   || <-> &&
    | <-> &
    < <-> >
```

and this automatic understand

```
False <-> True
FALSE <-> TRUE
# ... etc
```


# Usage
Set cursor on word or select word and press Super+Alt+x
